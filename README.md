# jeopardy-game-db
A Jeopardy-style quiz game built with Python and MongoDB, featuring real-time gameplay and NoSQL-based question storage

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Create a .env file: `cp .env.example .env`
4. Run MongoDB locally
5. Seed the database: `python seed.py`
